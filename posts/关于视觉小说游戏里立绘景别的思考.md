立绘的景别指的是作为人物的立绘，按照某种截取规则显示在程序窗体里的比例。这个概念参考自摄影，而因为游戏的程序窗体、人物展现内容和演出的表现存在一些特定的关系，故此特别说明并且立下规范。

# 景别规范

一般的，立绘会有五种景别：

| 镜头名称                     | 镜头英文名称 |
| ---------------------------- | ------------ |
| 膝盖下数公分能见到小腿的中景 | Calf Size    |
| 膝盖以上的中景               | Medium Size  |
| 腰部以上中景                 | Waist Size   |
| 胸部以上近景                 | Bust Size    |
| 特写（全脸）                 | Close Up     |

![景别图片](https://cdn.jsdelivr.net/gh/Zhuxb-Clouds/PicDepot/img/202407150107353.png)

# 解释：空间与距离感

对于不同的空间和不同的人物距离感，演出应该选择不同的景别：不同的距离代表着空间和人物关系的远近。

当人物占窗口比例太大，就会显得空间很挤。在一般的排版中，`Medium`（中景）由于人物在画面中占比较小，同屏可以轻松放下3-4个角色而不显得拥挤；而在`Waist`（腰部以上中景）的比例下，为了保证肢体动作的展现和留白，最多放置两个角色是比较合适的。至于`Bust`（胸部以上近景），由于截取位置较高，如果在画面边缘进行紧凑排版，有时也能容纳多人（如群像特写），但这需要极强的构图把控力，否则极易破坏空间的“正常感度”。

另外一个需要注意的点是：**留白与头顶距离**。除了角色立绘展现的内容，角色头顶到窗体顶部的距离也是一个值得斟酌的参数，但这一点更多的取决于背景图片的空间位置关系与人物之间的身高关系，因此此处不作过多的讨论。

# 深入解析：五种景别的演出运用

为了在实际开发中更准确地调用，我们需要明确这五种景别的具体“情绪”和“功能”：

### 1. Calf Size (小腿中景)
这是最接近“全身像”的景别。它通常用于角色**初次登场**，或者展现角色完整的服装设计、站姿和肢体语言。在这个景别下，玩家处于一个“旁观者”的客观视角，角色与玩家的心理距离较远，适合表现疏离感或处于广阔大场景中的状态。

### 2. Medium Size (膝盖以上中景)
这是视觉小说和2D游戏中最常用的景别之一。它保留了角色的大部分肢体动作（如双手抱胸、手插口袋等），同时又拉近了与玩家的距离。适合用于常规的日常对话、多人会议、或是角色之间保持标准“社交距离”的场景。

### 3. Waist Size (腰部以上中景)
当镜头推进到腰部以上时，玩家的视觉焦点会被强制集中在角色的上半身和面部表情上，肢体动作的展现退居其次。这是一种“私人距离”，适合用于一对一的深入交谈、角色表达个人观感、或是剧情开始进入小高潮时的演出。

### 4. Bust Size (胸部以上近景)
胸部以上的近景会占据大量的屏幕空间，角色仿佛“贴近”了屏幕。这种景别具有很强的压迫感或亲密感。通常用于：角色充满攻击性的质问、极其暧昧的凑近、或是极度悲伤/喜悦时的真情流露。

### 5. Close Up (全脸特写)
特写镜头直接切掉角色的身体，只保留面部（甚至只有眼部或嘴部）。在游戏中，特写不应该被滥用。它只在最关键的剧情节点出现：例如角色内心的瞬间崩塌、恍然大悟的震惊、或是极其危险的死亡凝视。特写能瞬间打破第四面墙，将玩家的沉浸感拉到满值。


立绘不仅仅是一张静态的插图，它是游戏演出的核心载体。通过规范化的“景别”管理，我们能用电影化的镜头语言去调动玩家的情绪。掌握好角色在窗体中的比例、距离与空间感，正是让一款游戏的叙事从“生硬的读屏”走向“沉浸式体验”的关键一步。
